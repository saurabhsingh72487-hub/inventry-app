# 📦 Inventory App (Express + MongoDB)

A simple inventory management backend built with **Node.js, Express, and MongoDB (Mongoose)**.

---

## 🚀 Features

- CRUD for:
  - Items
  - Categories
  - Publishers
- MongoDB relationships using ObjectId refs
- Admin password protection for update/delete
- Seed script for dummy data
- REST API structure

---

## 🛠️ Tech Stack

- Node.js
- Express.js
- MongoDB (Mongoose)
- dotenv
- nodemon
